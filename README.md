## Installing/Running the app

### Create a .env file

```
USERNAME="name you want displayed in project here"
MONGODB_URI="database creds here"
```

```bash
npm install
npm run dev
```

## Running Tests

```bash
npm run cy:open
```

### Formatting with Prettier

```bash
yarn format
```
